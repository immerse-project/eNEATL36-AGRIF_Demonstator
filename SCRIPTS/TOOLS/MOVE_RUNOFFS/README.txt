This tool helps creating the runoffs files for the parent and child grid of an AGRIF configuration

The current version may not be compatible with global grids.


##### HOW TO USE ###########

1 - Modify the paths in main_rnf_tool.sh

 
2 - Launch the script

- Locally : 

./main_rnf_tool.sh


- or on slurm

sbatch main_rnf_tool.sh
